How BloodLink Works

Register

Sign up as a donor or recipient with your blood type and location details.

Search

Find compatible donors near your location filtered by blood type.

Request

Send notifications to nearby donors for urgent blood requirements.

Connect

Connect with donors directly and arrange for blood donation.

Blood Type Compatibility

Type A+

Can receive from: A+, A-, O+, O-

Can donate to: A+, AB+

Type A-

Can receive from: A-, O-

Can donate to: A+, A-, AB+, AB-

Type B+

Can receive from: B+, B-, O+, O-

Can donate to: B+, AB+

Type B-

Can receive from: B-, O-

Can donate to: B+, B-, AB+, AB-

Type AB+

Can receive from: All blood types

Can donate to: AB+ only

Type AB-

Can receive from: A-, B-, AB-, O-

Can donate to: AB+, AB-

Type O+

Can receive from: O+, O-

Can donate to: A+, B+, AB+, O+

Type O-

Can receive from: O- only

Can donate to: All blood types
